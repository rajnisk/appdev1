## Troubleshooting Guide

### Common Issues & Quick Fixes:

**Database Issues:**
- If `db.create_all()` doesn't work: Delete `instance/taskmaster.db` and recreate
- If models not updating: Restart Flask server

**Template Issues:**
- If templates not found: Check `templates/` folder exists and is in project root
- If inheritance not working: Verify `{% extends "base.html" %}` is first line

**Auth Issues:**
- If login not working: Check password hashing matches (use `check_password_hash`)
- If session not persisting: Check `app.config['SECRET_KEY']` is set

**Import Errors:**
- If circular imports: Move models to separate file
- If module not found: Check virtual environment is activated
